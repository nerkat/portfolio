File Structure

The template package consists of the following files and directories:

  • `index.html`: the main html file of the template
  • `main.css: the main css file of the template
  • `main.js: the main js file of the template
  • `src/assets/images`: contains all the images used in the template
  • `src/assets/fonts`: contains all the fonts used in the template

How to Customize

  • To change the text, simply edit the HTML file in a text editor like Notepad++ or Sublime Text.
  • To change the images, replace the images in the `img/` directory.
  • To change the colors and styles, edit the stylesheets in the `css/` directory.
